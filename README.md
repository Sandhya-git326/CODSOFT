# CODSOFT Java Programming Internship

This repository contains the Java programming tasks completed as part of the "CODSOFT Java Development Internship Program". The projects demonstrate core Java programming concepts, object-oriented programming principles, user interaction, and API integration.

## Internship Tasks

### Task 1: Number Game

A console-based number guessing game where the user attempts to guess a randomly generated number within a limited number of attempts.

Features:

* Random number generation
* User input handling
* Attempt limitation
* Score tracking
* Replay option

Technologies Used:

* Java
* Scanner Class
* Random Class

### Task 2: Student Grade Calculator

A Java application that calculates the total marks, average percentage, and grade based on marks entered by the user.

Features:

* Accepts marks for multiple subjects
* Calculates total marks
* Calculates average percentage
* Determines grade based on performance
* Displays detailed results

Technologies Used:

* Java
* Conditional Statements
* Loops

### Task 3: ATM Interface

A console-based ATM simulation that performs basic banking operations.

Features:

* Check account balance
* Deposit money
* Withdraw money
* Validate transactions
* Exit functionality

Technologies Used:

* Java
* Object-Oriented Programming
* Exception Handling

### Task 4: Currency Converter

A Java application that converts currencies using real-time exchange rates fetched through an API.

Features:

* Select base currency
* Select target currency
* Enter amount to convert
* Fetch real-time exchange rates using API
* Display converted currency amount

Technologies Used:

* Java
* HTTP Connection
* JSON Parsing
* Exchange Rate API Integration
